Lab 4 — Статичні поля, властивості та методи класу
Клас BoardGame
Опис проєкту

Проєкт демонструє створення та застосування статичних полів, статичних властивостей, статичних методів, а також реалізацію методів Parse, TryParse і перевизначення ToString() у класі BoardGame (настільні ігри).

Програма містить консольне меню, яке дозволяє створювати, переглядати, знаходити та видаляти об’єкти, а також тестувати статичні методи класу.

Структура проєкту
Lab4/
│
├── Product.cs        // Базовий клас
├── BoardGame.cs      // Основний клас із статичними членами
└── Program.cs        // Меню та взаємодія з користувачем

Можливості класу BoardGame
Властивості гри:

назва

жанр

мінімальна/максимальна кількість гравців

тривалість партії

рекомендований вік

ціна

видавець

Перевантажені конструктори:

без параметрів

з одним параметром

з декількома параметрами

повна ініціалізація

Методи:

DisplayInfo() (у кількох варіантах)

PlayDemo() — поведінка об’єкта

ToString() — рядкове представлення об’єкта

Статичні поля та властивості:

Counter — кількість створених об’єктів

DiscountRate — глобальна знижка (0–50 %)

ClubName — назва клубу настільних ігор

Статичні методи:

GetDiscountedPrice(BoardGame game)

Parse(string s)

TryParse(string s, out BoardGame game)

Формат рядка для Parse/TryParse
Назва;Жанр;MinPlayers;MaxPlayers;Тривалість;MinAge;Ціна;Видавець


Приклад:

Каркассон;Стратегія;2;5;60;8;800;Hobby World

Приклади використання
Parse()
string s = "Монополія;Економічна;2;6;90;8;1000;Hasbro";
BoardGame game = BoardGame.Parse(s);

TryParse()
if (BoardGame.TryParse(s, out BoardGame game))
{
    Console.WriteLine("Успішно створено");
}

Статичний метод
decimal discounted = BoardGame.GetDiscountedPrice(game);
