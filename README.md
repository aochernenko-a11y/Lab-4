Статичні поля, властивості та методи класу
Тема: Клас BoardGame (Настільні ігри)
Опис проекту

Проєкт реалізує предметну область настільних ігор.
Розроблено повноцінний клас BoardGame, який містить:

основні властивості гри (назва, жанр, кількість гравців, тривалість, вік, ціна, видавець);

статичні поля та властивості, спільні для всіх об’єктів класу;

статичні методи (включно з Parse та TryParse);

перевизначений метод ToString() у форматі, сумісному з Parse/TryParse;

перевантажені конструктори та перевантажені методи виведення;

поведінку об’єкта (метод PlayDemo).

Проєкт демонструє роботу зі статичними членами класу, механізмами перетворення рядків та об’єктів, а також роботу меню для взаємодії з користувачем.

Основні можливості
Статичні поля та властивості

Counter — лічильник коректно створених об’єктів BoardGame

DiscountRate — глобальна знижка (0–50 %)

ClubName — назва клубу настільних ігор

Статичні методи

GetDiscountedPrice(BoardGame) — обчислення ціни з урахуванням знижки

Parse(string) — створення гри з рядка, генерує виключення у випадку помилок

TryParse(string, out BoardGame) — безпечний варіант парсингу

Перевизначений метод

ToString() — повертає рядок у форматі, який повністю підтримується Parse і TryParse

Приклад роботи статичних методів
BoardGame.DiscountRate = 0.15m;

Приклад використання Parse()
string data = "Монополія;Економічна;2;6;90;8;1000;Hasbro";
BoardGame g = BoardGame.Parse(data);

Приклад використання TryParse()
if (BoardGame.TryParse(data, out BoardGame g))
{
    Console.WriteLine("Успішно створено об’єкт!");
}
else
{
    Console.WriteLine("Помилка формату.");
}
